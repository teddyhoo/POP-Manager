README: PET OWNER PORTAL SCRIPTS

The package consists of the following elements:

  • jquery-ajax.html: this file initiates jquery ajax request async to locally running instance of the node js server (app.js)
  • pop.js: LeashTime pet owner portal (pop) library module of object prototypes, parsing JSON, unit-test populating object representations, helper functions
  • app.js: NODE JS server app; run from command line with node app.js; reads visit.json file and returns it in response to AJAX request
  • visit.json: raw JSON file, formatted and saved out to file system for use by node server app

Install node js. HERE. Get the macOS Installer, double click and node js will install as well as npm [node package manager]. Unzip the files and put in same directory. Then go to that directory and type node app.js from the command line terminal. Then open hte jquery-ajax.html file.
